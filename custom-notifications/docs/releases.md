# Releases

`mpt-extension-contrib-custom-notifications` is released independently with a
`notifications-<version>` tag.

Release this package only when its own API, a channel, or a declared dependency
(including an extra) changes. All channels ship in the one distribution and
share its version; adding or changing a channel is a release of this package.

Follow the repository-wide [release workflow](../../docs/releases.md).
